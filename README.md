# ClipShader
UnityのShader Graphを利用した3Dモデルの特定領域を動的にクリッピング（切り取り/非表示化）するシェーダーのデモンストレーションプロジェクトです。

## 概要
特定のオブジェクト（例：Cube）と3Dモデルが重なった部分を非表示にすることができます。
断面表示の効果を実現できます。
